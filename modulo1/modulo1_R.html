Respuestas del Examen

Preguntas de Falso y Verdadero:

  • Los ataques de phishing son una amenaza cibernética común. (Verdadero)
  • El robo de identidad no es considerado una amenaza cibernética. (Falso)
  • Los virus informáticos son una forma de malware. (Verdadero)
  • Los ataques de fuerza bruta se enfocan en descifrar contraseñas mediante intentos repetidos. (Verdadero)
  • Los ataques de denegación de servicio (DDoS) se utilizan para saturar un sistema o red y hacerlo inaccesible. (Verdadero)
  • Las vulnerabilidades de seguridad solo se encuentran en sistemas operativos, no en software. (Falso)
  • El cifrado es una medida de seguridad que protege los datos mediante la codificación. (Verdadero)
  • Las botnets son grupos de computadoras comprometidas que pueden ser controladas de forma remota. (Verdadero)
  • El ransomware es una forma de ataque que busca secuestrar los datos de un sistema hasta que se pague un rescate. (Verdadero)
  • La auditoría de seguridad se utiliza para evaluar la eficacia de las medidas de seguridad implementadas. (Verdadero)

Preguntas de Completar:

  • La autenticación es un proceso que verifica la IDENTIDAD de un usuario o entidad.
  • El hacking ético se realiza con el objetivo de identificar y corregir VULNERABILIDADES en sistemas y redes.
  • El firewall es una medida de seguridad que controla el FLUJO de tráfico entre redes.
  • Los exploits son técnicas o herramientas utilizadas para aprovechar VULNERABILIDADES en sistemas o software.
  • La respuesta a incidentes implica detectar, investigar y RESOLVER los incidentes de seguridad.
  • La intrusión se refiere a la ACCESO no autorizada a un sistema o red.
  • El ransomware es un tipo de MALWARE que cifra los datos y exige un rescate para su liberación.
  • La vulnerabilidad es una debilidad o fallo en un sistema que puede ser EXPLOTADA por atacantes.
  • La botnet es una red de computadoras controlada de forma REMOTA para llevar a cabo actividades maliciosas.
  • La auditoría de seguridad evalúa la EFICACIA de los controles de seguridad implementados.

Preguntas de Selección Múltiple:

  1. ¿Qué es el malware? - a) Un tipo de virus informático.
  2. ¿Cuál de las siguientes amenazas cibernéticas busca obtener información confidencial mediante la suplantación de identidad? - a) Ataque de phishing.
  3. ¿Cuál de las siguientes medidas de seguridad protege una red o sistema limitando el acceso no autorizado? - c) Firewall.
  4. ¿Qué es un exploit? - d) Una herramienta utilizada para aprovechar vulnerabilidades.
  5. ¿Cuál de las siguientes actividades implica el hacking ético? - c) Realizar pruebas de seguridad en redes.
  6. ¿Cuál de las siguientes amenazas cibernéticas se enfoca en interrumpir el funcionamiento normal de un sistema o red? - c) Ataque de denegación de servicio (DDoS).
  7. ¿Qué implica la auditoría de seguridad? - b) La evaluación de los controles de seguridad implementados.
  8. ¿Cuál de las siguientes medidas de seguridad se utiliza para proteger la integridad y confidencialidad de los datos? - c) Cifrado.
  9. ¿Qué es el robo de identidad? - d) La apropiación de la información personal de alguien para cometer fraudes.
  10. ¿Cuál de las siguientes amenazas cibernéticas se centra en el secuestro de datos y la exigencia de un rescate? - c) Ransomware.
